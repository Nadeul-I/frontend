## EnjoyTrip

소개 및 개요

- 프로젝트 기간 : 2023.05 ~ 2023.06

[프로젝트 설명]

- 여행지에 대한 정보를 공유할 수 있는 플랫폼입니다.
- 사용자들은 여행지를 조회할 수 있습니다.
- 사용자들은 게시판에 여행지에 대한 후기를 남길 수 있습니다.

## 기술 및 개발 환경

Front-end : Vue.js, Vuex

Back-end : Java, Spring boot

DB : MySQL

## 주요 기능

회원가입/로그인

- 로그인
- 회원가입
